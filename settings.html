GitHub 연결 설정

여기서 한 번만 설정해두면, 브랜드를 등록하거나 수정할 때마다 이 사이트가 자동으로 GitHub 저장소에 커밋합니다.

1. 개인 액세스 토큰(Fine-grained token) 만들기

1
github.com/settings/tokens 로 이동해서 "Generate new token"을 누르세요.
2
Repository access에서 "Only select repositories"를 고르고, 이 사이트가 올라간 저장소(예: my-homepage)를 선택하세요.
3
Permissions → Repository permissions → Contents를 "Read and write"로 바꾸세요.
4
토큰을 생성하고, 화면에 뜬 토큰 문자열을 복사해서 아래 입력칸에 붙여넣으세요. (토큰은 생성 직후에만 보이니 꼭 복사해두세요.)

⚠️ 이 토큰은 이 브라우저의 저장공간(localStorage)에만 저장됩니다. 본인만 쓰는 컴퓨터에서 사용하세요. 공용 컴퓨터라면 사용 후 아래 "연결 정보 삭제" 버튼으로 지워주세요.
